# E-Commerce Sales Analytics Dashboard

## Overview

This project analyzes retail sales data using Python, SQL, and Power BI to generate business insights related to revenue, profitability, customer behavior, and sales forecasting.

The project includes data cleaning, exploratory data analysis, customer segmentation, SQL-based reporting, forecasting, and an interactive Power BI dashboard.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* SQLite
* SQL
* Power BI

---

## Features

### Executive Dashboard

* Total Sales
* Total Profit
* Total Orders
* Total Customers
* Monthly Sales Trend
* Revenue by Region
* Sales by Category

### Sales Analytics

* Top Products by Revenue
* Profit by Category
* Monthly Profit Trend

### Customer Analytics

* Customer Segment Distribution
* Top Customers by Revenue
* Customer Segmentation Analysis

### Forecasting

* Sales Forecast Trend
* Future Sales Prediction
